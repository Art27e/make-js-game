#### Code Description
A classic Arkanoid game built with vanilla JavaScript and HTML5 Canvas. <br /> Actively developed in 2026 ‚ÄĒ planned updates include visual redesign, sprite/sound integration, bug fixes, and migration from Canvas to pure HTML/CSS/JS.

#### Game Updates
### 1.0.0
- Refactored ball collision detection.
- Redesigned the game visuals.
- Added start screen and pause functionality.
- Increased ball speed.
- Introduced a durability-based brick system:
  - Light bricks break in 1 hit.
  - Normal bricks break in 2 hits.
  - Heavy bricks break in 3 hits.
